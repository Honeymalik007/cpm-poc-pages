Duplicate ID — forces XPath fallback in scraper

The element above contains a malicious id attribute. When changedetection.io scrapes this page via Playwright and stores the visual selector data, the id value is written unsanitized into elements.deflate. On hover over the Visual Selector canvas, visual-selector.js:253 assigns the raw XPath to innerHTML, triggering XSS in the operator's browser at the changedetection.io origin.
